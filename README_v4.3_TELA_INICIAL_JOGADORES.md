# v4.3_player_home_bracket_button

Esta versão transforma a área de eventos/participação em tela inicial do painel.

## O que mudou

- O painel abre primeiro na tela de eventos para jogadores/capitães.
- O chaveamento atual continua existindo.
- O chaveamento agora é acessado pelo botão `Chaveamento` no topo ou `Ver chaveamento` na tela inicial.
- Dentro da tela de chaveamento existe o botão `Voltar aos eventos`.
- A página `/pages/player-event-views.html` continua existindo como prévia das 5 versões, mas sem a imagem gigante solta no topo.
