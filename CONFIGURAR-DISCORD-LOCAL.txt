CONFIGURAR LOGIN DISCORD LOCAL — VOID ARENA

1. No Discord Developer Portal, entre na aplicação do bot.

2. Como o token/client secret foram colados no chat, faça:
   - Bot > Reset Token
   - OAuth2 > Reset Client Secret

3. Copie o arquivo:
   Windows CMD:
     copy .env.example .env

   Linux/Mac/Git Bash:
     cp .env.example .env

4. Abra o arquivo .env e preencha:
   DISCORD_TOKEN=token novo do bot
   CLIENT_ID=Application ID / Client ID
   DISCORD_CLIENT_ID=Application ID / Client ID
   DISCORD_CLIENT_SECRET=Client Secret novo
   DISCORD_CALLBACK_URL=http://localhost:3000/auth/discord/callback

5. No Discord Developer Portal > OAuth2 > Redirects, adicione exatamente:
   http://localhost:3000/auth/discord/callback

6. Rode:
   npm install
   npm start

7. Abra:
   http://localhost:3000

IMPORTANTE:
Não envie .env para ninguém.
Não suba .env no GitHub.
Não coloque token/secret dentro de ZIP compartilhado.
