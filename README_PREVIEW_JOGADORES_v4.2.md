# v4.2_player_event_views_preview

Esta versão adiciona uma página de prévia para escolher qual layout fica melhor para jogadores, capitães e donos de times.

Abra:

```txt
http://localhost:3000/pages/player-event-views.html
```

ou, no Render:

```txt
https://SEU-SERVICO.onrender.com/pages/player-event-views.html
```

A página tem 5 versões:
1. Central de Eventos
2. Painel do Capitão
3. Agenda de Campeonatos
4. Lobby Competitivo
5. Inscrição Guiada

Nada foi aplicado ainda como tela oficial. É só uma prévia visual para escolher o melhor modelo.
